# 🎯 Bubble Hit Game

A fun and interactive **Bubble Hit Game** built using **HTML, CSS, and JavaScript**. The goal is simple: hit the correct number bubble before the timer runs out and score as high as possible!

## 🕹️ How to Play

* You’ll see a **target number (Hit Number)** at the top.
* Click on the bubble that matches the target number.
* Each correct hit increases your score.
* The game runs on a timer — score as much as you can before time ends!

* 🎯 Random number generation for each round
